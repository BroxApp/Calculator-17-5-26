# Simple Web Calculator

A functional and responsive web-based calculator that allows users to perform basic arithmetic operations with a clean, modern interface.

## Features
- Basic operations: Addition, Subtraction, Multiplication, and Division.
- Support for percentage calculations.
- Full keyboard support (Numbers, Operators, Enter for '=', Escape for 'C', Backspace).
- Responsive design using CSS Grid.
- Interactive UI with hover and active button states.

## How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use the on-screen buttons or your keyboard to perform calculations!

## Technologies Used
- HTML5 / CSS3 / Vanilla JavaScript
